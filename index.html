Game Rules & Description

Objective:

Create a continuous circular railway line that connects all required tiles.

How to Play:

  • Left Click: Place selected rail on a tile
  • Right Click: Rotate placed rail 90Â° clockwise
  • Mountains & Bridges: Must have rails placed on them
  • Oases: Cannot have rails (trains must go around)
  • Empty Tiles: Optional - use as needed for your route

Rail Types:

  • Straight Rail: For empty tiles (can be rotated)
  • Curve Rail: For empty tiles (can be rotated)
  • Bridge Rail: Only for bridge tiles
  • Mountain Rail: Only for mountain tiles

Win Condition:

All rails must form one continuous loop with proper connections!
